Есть два способа запуска проекта:

1. Просто перейти по ссылке: https://pixel-perfect-build-08.lovable.app

2. Запустить локально:

2.1 Скачайте код проекта через GitHub
   
2.2 Установите Node.js
Скачайте и установите Node.js с официального сайта: https://nodejs.org/ (выберите LTS версию)

2.3 Откройте терминал в папке проекта
Windows: Shift + правая кнопка мыши в папке → "Открыть окно PowerShell здесь"
Mac/Linux: Откройте Terminal и перейдите в папку: cd путь/к/папке

2.4 Установите зависимости
npm install

2.5 Запустите проект
npm run dev

После запуска вы увидите сообщение типа:

Local: http://localhost:8080
Откройте этот адрес в любом браузере (Chrome, Firefox, Safari и т.д.)

Важно: Проект будет работать только пока открыт терминал. Чтобы остановить — нажмите Ctrl+C в терминале.

Примечание: Если при после ввода команды "npm install" выдаёт ошибку типа:евозможно загрузить файл C:\путь\к\папке, так как выполнение сценариев отключено в этой системе. Для получения дополнительных сведений см. about_Execution_Policies по адресу https:/go.microsoft.com/fwlink/?LinkID=13 5170
Тогда нужно будет зайти в поиск Windows и запусть Windows PowerShell от имени администратора, после чего прописать в поле ввода данную команду: "Set-ExecutionPolicy Unrestricted -Scope CurrentUser" и нажать "Y" для подтверждения.
После этого повторить действия описанные в подпунктах выше.
